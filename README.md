# Research

This repository stores long-form research papers and their compiled reading
copies. Each paper lives in its own directory and uses the same file names:

- `paper.tex` is the editable LaTeX source.
- `paper.pdf` is the compiled reading copy.
- `metadata.yaml` gives a short machine-readable summary.
