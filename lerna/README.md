# Lerna Protocol

This directory contains the revised Lerna design-note draft:

- `paper.tex` is the editable source.
- `paper.pdf` is the compiled reading copy after TeX compilation.
- `docs/` contains supporting adaptation notes and mapping tables.

Current scope: ADA-only V0 inside an open Hydra Head. Native assets,
reduced-signature non-interference, and V1 L1 claim UTxOs are documented as
future-work or benchmark tracks, not as deployable claims.
